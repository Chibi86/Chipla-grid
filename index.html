Chipla-grid

Welcome to the theme Chipla-grid!
Less code
Grid system
Vector icons

This is a theme made by Rasmus Berg for Chipla-MVC for the cource PHPMVC at college BTH in sweden.

It's coded in less-code and got some external libaries for make cross-browser, grid-based and get vector icons.

Has possibly be in a standalone project but it's at first part made for Chipla-MVC which is based on Anax-MVC so it should work for that to.

Contains:

  • Less
    • lessphp v0.4.0
  • Grid system
    • Semantic.gs v1.2
  • Cross-Browser
    • Normalize v4.1.1
    • Less mixins from Lydia v0.3.x
  • Vector icons
    • Font Awesome v4.6.3
  • Responsive
  • Anax-MVC support
  • Chipla-MVC support
